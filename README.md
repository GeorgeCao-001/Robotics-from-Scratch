# Robotics-from-Scratch

从零开始构建一个人脸跟随机器人小车。

## 项目概述

本项目是一个基于树莓派（Raspberry Pi）和 Arduino Uno 的人脸跟随机器人系统。通过摄像头实时检测人脸位置，控制云台跟踪人脸，同时驱动小车自动跟随目标，保持安全距离。

### 核心功能

- **实时人脸识别**：使用 MediaPipe 进行人脸检测，帧率 >= 15 FPS
- **云台自动跟踪**：双轴舵机云台，自动调整角度使人脸居中
- **小车自动跟随**：基于 PID 控制，根据人脸距离动态调整车速
- **安全机制**：心跳超时保护、低电压保护、指令验证

### 技术亮点

- **分布式架构**：树莓派负责视觉与决策，Arduino 负责实时控制
- **模块化设计**：视觉、规划、控制分离，易于调试和扩展
- **JSON 通信**：基于 UART 串口的标准化通信协议
- **PID 闭环控制**：精确控制跟随距离和角度

## 系统架构

### 硬件架构

```
USB摄像头 → 树莓派 --UART--> Arduino Uno --> L298N --> 4WD底盘
                ↓                          ↓
           云台舵机(PWM)              电池监测(ADC)
```

**组件说明**：
- **树莓派**：运行视觉算法和决策逻辑（Python）
- **Arduino Uno**：实时电机控制和传感器读取（C++）
- **4WD 底盘**：四轮驱动，差速转向
- **L298N**：双路电机驱动模块
- **双轴云台**：水平（pan）+ 垂直（tilt）舵机

### 软件架构

```
树莓派（高层）                Arduino Uno（底层）
├─ 视觉模块                    ├─ 串口解析
│   └─ 人脸检测                    └─ JSON解析
│   └─ 坐标计算                ├─ 电机控制
├─ 规划模块                    │   ├─ PWM生成
│   ├─ PID控制器                 ├─ 轮速分配
│   ├─ 距离估计              ├─ 传感器读取
│   └─ 指令生成                    └─ 电池电压
├─ 硬件接口                    └─ 状态反馈
│   ├─ 云台控制
│   └─ 串口通信
└─ main.py
```

### 通信协议

基于 UART 串口，波特率 115200，JSON 格式：

**运动控制指令**：
```json
{
  "cmd": "move",
  "v": 0.5,    // 线速度 (m/s)
  "w": 0.1     // 角速度 (rad/s)
}
```

**云台控制指令**：
```json
{
  "cmd": "gimbal",
  "pan": 15.0,   // 水平角度 (度)
  "tilt": -10.0  // 垂直角度 (度)
}
```

**状态响应**：
```json
{
  "status": "ok",
  "battery": 87,
  "error": null
}
```
