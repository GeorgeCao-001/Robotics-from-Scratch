# AGENTS.md (Raspberry Pi)

## 角色

负责：
- 视觉处理
- 决策制定
- 指令生成

---

## 模块

### 视觉模块
- 输入：摄像头帧
- 输出：结构化数据

示例：

```json
{
  "target_x": 120,
  "target_y": 80,
  "width": 150
}
```

- `target_x`, `target_y`：目标中心在画面中的像素坐标
- `width`：目标宽度（用于距离估计）

---

### 云台模块
- 输入：目标位置偏移
- 输出：舵机角度指令

示例：

```json
{
  "pan": 15.0,
  "tilt": -10.0
}
```

**核心逻辑**：如果人脸在画面左侧，增加偏转角（pan），使人脸回到画面中心。

**控制策略**：
- 水平舵机（pan）：控制左右转动
- 垂直舵机（tilt）：控制上下转动
- 采用简单的比例控制（P控制）

---

### 决策模块
- 输入：视觉输出（目标位置、宽度）
- 输出：运动指令

示例：

```json
{
  "cmd": "move",
  "v": 0.3,
  "w": -0.2
}
```

**PID 控制策略**：

线速度控制（基于距离）：
```
v = Kp_distance × (target_distance - actual_distance)
```

角速度控制（基于水平偏移）：
```
w = Kp_angle × (target_x - image_center_x)
```

参数说明：
- `target_distance`：期望跟随距离（例如：1.0米）
- `actual_distance`：根据目标宽度估算的实际距离
- `image_center_x`：画面中心 x 坐标
- `Kp_distance`, `Kp_angle`：比例系数（需要调参）

## 约束

- 禁止直接控制电机
- 禁止包含 ESP32 特定逻辑

---

## 性能

- 目标帧率 >= 15 FPS
- 最小化延迟

---

## 故障处理

- 如果视觉模块故障 → 发送 STOP 指令
